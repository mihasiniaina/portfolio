##  Getting Started

### Prerequisites
* Docker & VS Code (if using the Dev Container)
* Node.js (v18+ or later) & npm (if running locally)

### Setup with VS Code Dev Containers (Recommended)
1. Clone this repository.
2. Open the project folder in VS Code.
3. Click **"Reopen in Container"** when prompted (or open the Command Palette with `Ctrl+Shift+P` and select `Dev Containers: Reopen in Container`).
4. The environment, dependencies, and extensions will install automatically!

### Local Installation (Alternative)
```bash
git clone git@github.com:mihasiniaina/portfolio.git

npm install

npm run dev 

```
